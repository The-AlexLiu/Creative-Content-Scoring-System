# 内容结构评分细则

这份评分细则用于统一团队对短视频、达人视频和广告素材的判断口径。每个模块评分范围为 0-10 分。

## 1. Hook

Hook 主要判断内容开头是否具备停留能力。

优秀 Hook 常见特征：

- 明确痛点
- 情绪冲突
- 结果前置
- 强反差
- 悬念引导
- 场景代入
- 直接展示问题或效果

评分建议：

| 分数 | 判断标准 |
|---:|---|
| 8-10 | 开头 1-3 秒有明显停留点，痛点、结果或反差非常清晰 |
| 5-7 | 有一定吸引力，但信息不够尖锐或节奏偏慢 |
| 0-4 | 开头平淡、铺垫过长、无法快速解释为什么要继续看 |

## 2. 效果证言

效果证言是当前评分中权重最高的模块，主要判断内容是否提供了足够可信的效果证明。

优秀证言常见特征：

- 用户真实反馈
- 前后对比
- 使用过程展示
- 使用结果展示
- 身份背书
- 权威背书
- 具体细节描述

评分建议：

| 分数 | 判断标准 |
|---:|---|
| 8-10 | 证言具体、可信，有结果展示或清晰前后对比 |
| 5-7 | 有反馈或演示，但细节不足，可信度一般 |
| 0-4 | 只有主观夸赞，无过程、无结果、无可验证信息 |

## 3. 卖点讲解

卖点讲解判断内容是否把产品价值讲清楚。

好的卖点讲解需要做到：

- 功能点明确
- 用户收益清晰
- 解释足够通俗
- 不只是堆参数
- 能把产品功能和用户痛点连接起来

评分建议：

| 分数 | 判断标准 |
|---:|---|
| 8-10 | 卖点清楚，能解释产品为什么对目标用户有用 |
| 5-7 | 提到了卖点，但表达偏泛或和痛点连接不强 |
| 0-4 | 几乎没有讲清产品价值，用户看完仍不知道为什么要买 |

## 4. CTA 号召

CTA 判断内容是否有清晰的下一步行动引导。

常见 CTA：

- 点击链接
- 查看主页
- 评论互动
- 收藏
- 私信
- 下单
- 关注账号
- 参与活动

评分建议：

| 分数 | 判断标准 |
|---:|---|
| 8-10 | 行动指令清晰，与内容场景自然衔接 |
| 5-7 | 有行动提示，但不够明确或不够顺滑 |
| 0-4 | 没有 CTA，或 CTA 与内容割裂 |

## 投放判断建议

| 分档等级 | 建议动作 | 投放注意点 |
|---|---|---|
| 优先验证 | 小预算测试、重点测新、推荐达人复拍 | 优先验证 CTR、CVR、CPA、ROAS，不要只看单条素材主观质量 |
| 观察优化 | 观察数据、优化复拍 | 找出最低分模块，优先改 Hook 或证言 |
| 不建议直投 | 不建议直接投放，先二创或淘汰 | 除非有极低制作成本或特殊受众价值，否则避免占用测试预算 |
