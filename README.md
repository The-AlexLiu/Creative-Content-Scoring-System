# Creative Content Scoring System

**内容结构评分系统：把“这条素材好不好”拆成可复盘的判断标准**

做广告素材和短视频内容时，最容易出现的问题不是大家没有判断，而是每个人的判断标准不一样。

投放同学可能会说：“这条开头不够抓人。”  
内容同学可能会说：“达人表达挺自然的。”  
品牌同学可能会担心：“卖点是不是讲得太硬。”  
老板最后可能只问：“这条到底值不值得投？”

这些判断都不是错的，但如果没有统一口径，素材复盘很容易变成感觉讨论。看起来大家都在评价同一条视频，其实每个人看的维度不一样。

所以我做这套评分系统时，想解决的不是“用一个分数决定素材生死”，而是把原本模糊的素材判断拆开：

- 开头有没有让用户停下来？
- 证言和效果有没有可信度？
- 卖点有没有讲清楚？
- 有没有自然的行动引导？
- 这条素材适合直接投放、二创，还是应该淘汰？

它更像一个复盘工具，而不是一个绝对裁判。

## 我为什么要做这套评分

在真实投放里，素材判断经常会遇到几个问题。

第一，很多素材上线前只能靠经验筛。经验当然有价值，但如果判断没有被拆成具体维度，就很难传递给团队里的其他人。

第二，素材复盘容易只看结果。比如一条视频 ROAS 高，大家就说它是好素材；一条视频 CPA 高，大家就说它不行。但如果不回头拆内容结构，就很难知道到底是 Hook 好、证言强、卖点清楚，还是只是刚好吃到了某个受众或投放窗口。

第三，达人素材很难沉淀。一个达人视频表现好，如果只记录“这个达人不错”，下次复拍还是容易重新靠感觉写 Brief。更好的方式是记录它到底强在哪里：是开头场景强、身份背书强、前后对比强，还是 CTA 更自然。

所以这套系统的目标很简单：

> 把“我觉得这条素材不错”变成“这条素材在哪些结构上更值得继续测试”。

---

## 我的核心判断

一条短视频广告是否值得继续测试，不能只看它有没有高级感，也不能只看它是不是拍得漂亮。

我更关心它有没有完成 4 件事：

1. **先让人停下来**：用户愿不愿意继续看前几秒。
2. **再让人相信**：内容有没有真实反馈、效果展示或身份背书。
3. **把价值讲清楚**：用户能不能听懂产品为什么和自己有关。
4. **给出下一步**：看完以后用户知不知道该点击、评论、下单、私信还是继续了解。

对应到评分里，就是 4 个模块：

| 模块 | 权重 | 我主要看什么 |
|---|---:|---|
| Hook | 30% | 开头是否有停留点，痛点、反差、结果或情绪是否足够明确 |
| 效果证言 | 40% | 用户反馈、前后对比、使用过程、结果展示、身份背书是否可信 |
| 卖点讲解 | 20% | 产品价值是否讲清楚，功能和用户痛点是否连接起来 |
| CTA 号召 | 10% | 行动引导是否自然、明确，是否和内容场景顺滑衔接 |

我把“效果证言”的权重放得最高，是因为很多转化型素材真正打动用户的，不是品牌自己说自己好，而是用户看到一个具体、可信、和自己有关的结果。

## 总分怎么算

每个模块按 0 到 10 分评分，最终得到一个 0 到 10 分的综合分。

```text
总分 = Hook * 0.3 + 效果证言 * 0.4 + 卖点讲解 * 0.2 + CTA * 0.1
```

这个分数不是为了替代投放数据，而是为了让投放前筛选和投放后复盘有一个统一语言。

| 等级 | 分值区间 | 我会怎么理解 | 建议动作 |
|---|---:|---|---|
| 优保 | 7.0-10 | 内容结构完整，有继续测试或复拍价值 | 主力投放、重点测新、推荐达人复拍 |
| 普通 | 4.0-6.9 | 有亮点，但存在明显弱项 | 常规测新、局部优化、作为二创素材 |
| 低质 | 0-3.9 | 缺少有效结构，投放前风险较高 | 不建议直接投放，进入二创或淘汰 |

这里的“优保”不是说一定能跑出好结果，而是说它的内容结构值得被优先验证。真正能不能放量，还要看 CTR、CVR、CPA、ROAS、评论质量、受众反馈和样本量。

## 我会怎么使用这套系统

### 第一步：先把素材拆成结构

我不会一上来就问“这条好不好”，而是先把视频拆成几个问题：

| 问题 | 对应评分 |
|---|---|
| 前 1 到 3 秒有没有理由让人停下？ | Hook |
| 是否出现真实使用者、过程、结果或前后对比？ | 效果证言 |
| 用户看完是否知道产品解决什么问题？ | 卖点讲解 |
| 结尾或中段有没有自然的下一步动作？ | CTA |

这样做的好处是，即使一条素材总分一般，也能知道它到底差在哪里。比如 Hook 不差，但证言弱，就不一定要重拍整条视频，可能只需要补真实反馈或使用结果。

### 第二步：再判断适合怎么处理

评分不是为了简单淘汰，而是为了决定下一步动作。

| 评分结果 | 我会优先考虑 |
|---|---|
| Hook 高、证言高 | 优先小预算测试，观察 CTR、CVR 和评论反馈 |
| Hook 高、证言低 | 保留开头，补充真实使用过程或结果展示 |
| 证言高、Hook 低 | 用原素材做二创，重剪开头 |
| 卖点高、CTA 低 | 保留讲解，重写行动引导 |
| 全部偏低 | 不建议占用测试预算 |

### 第三步：把评分和投放数据放在一起看

内容评分只能解释“结构质量”，不能直接等于投放结果。

所以复盘时我会把评分和这些数据一起看：

- 花费；
- 展示；
- 点击；
- CTR；
- CPC；
- CPM；
- CVR；
- CPA；
- ROAS；
- 评论和私信反馈；
- 素材疲劳速度；
- 归因窗口和投放日期。

如果一条素材评分高但投放不好，我会继续追问：是不是受众不匹配、落地页没接住、价格没有吸引力，还是样本太少？  
如果一条素材评分不高但跑得不错，我也会回头看：它是不是有某个被评分表低估的场景、情绪或人群洞察？

## 快速使用

如果只是想马上开始给素材打分，可以直接用 CSV 模板。

### 方式 1：用表格评分

1. 打开 [`templates/content_scoring_template.csv`](./templates/content_scoring_template.csv)。
2. 点击 GitHub 页面右上角的 `Download raw file` 下载 CSV。
3. 导入 Google Sheets 或 Excel。
4. 为每条素材填写 `视频编号`、`达人/视频链接`、`Hook 30%`、`证言 40%`、`卖点 20%`、`CTA 10%` 和 `评语建议`。

建议评分表结构：

| 视频编号 | 达人/视频链接 | Hook 30% | 证言 40% | 卖点 20% | CTA 10% | 总分 | 分档等级 | 评语建议 |
|---|---|---:|---:|---:|---:|---:|---|---|
| #001 | 示例链接 | 8 | 9 | 7 | 6 | 8.0 | 优保 | Hook 抓人，证言清晰，CTA 可强化 |
| #002 | 示例链接 | 4 | 3 | 2 | 0 | 2.8 | 低质 | 无结构，无有效演示 |

Excel / Google Sheet 总分公式：

```text
=C2*0.3+D2*0.4+E2*0.2+F2*0.1
```

分档公式：

```text
=IFS(G2>=7,"优保",G2>=4,"普通",G2<4,"低质")
```

### 方式 2：用脚本批量算分

如果已经有一批素材评分数据，可以用脚本自动计算 `总分` 和 `分档等级`：

```bash
python3 scripts/score_content.py examples/example_input.csv examples/example_scored.csv
```

脚本会自动输出：

- `总分`
- `分档等级`
- `错误信息`，例如评分为空、评分不是数字、评分超出 0-10

## 后续可以继续优化

这套评分系统目前还是一个基础版本，后续可以继续往几个方向扩展：

- 按不同平台调整权重，例如 Meta、TikTok、小红书、YouTube Shorts；
- 按不同品类调整评分标准，例如美妆、服饰、家居、电子产品、健康类产品；
- 增加素材标签体系，例如痛点、场景、证言类型、卖点类型、CTA 类型；
- 接入投放数据，分析评分和 CTR、CVR、CPA、ROAS 的关系；
- 增加 AI 自动初评，再由人工复核；
- 建立素材数据库和可视化看板，沉淀高分素材模式。

## 我希望作品集传达的能力

这个项目想表达的不是“我会做一个评分表”，而是：

1. 我能把主观的内容判断拆成结构化标准。
2. 我能把素材生产、投放测试和复盘沉淀连起来。
3. 我知道评分不能替代投放数据，但可以让测试前后的讨论更清楚。
4. 我能把达人视频、广告素材、复拍 Brief 和二创判断放进同一套工作流里。
5. 我能用表格和脚本把方法做成可复用工具，而不是只停留在经验描述。
